Slidebars Theme

The Slidebars theme includes additional markup and styling for use with Slidebars. Other than initial colours for indication, the Slidebars theme has no font, element or tag alterations that would alter any of your own styling. Classes used are prefixed with sb- to avoid any conflict, however you are free to change this.

Features

  • Navbar
  • Navbar menu items
  • Slidebar menu items (open left Slidebar)
  • Slidebar widgets (open right Slidebar)

Usage

You can either inspect the markup in the files included in this example, or see http://plugins.adchsm.me/slidebars/theme.php for a breakdown.
